markdown syntax

Markdown 语法

Markdown 标题

使用#号表示1-6级标题，一级标题对应一个#号。

Markdown 段落格式

段落的换行是使用两个以上的空格加回车
也可使用空行来表示换行

👆面有空行！

Markdown 字体

斜体文字 ※文字※
斜体文字 或者下划线
粗体文字 ※※文字※※（或者下划线）
粗斜体文字 ※※※文字※※※（或者下划线）

Markdwon 分割线

三个以上的星号减号或者下划线，行内除了空格外不能有其他字符
- 删除线
Hello,World 两条波浪线
- 下划线
使用HTML的标签实现
Hello World!
- 脚注
> 格式[^要注明的文本]
[^FirstCode]: Hello,World!

Markdown 列表

支持有序列表以及无序列表
无序列表使用*号、+号以及-号作为列表标记
有序列表使用数字并加上.号表示

  1. Hello
  2. World

Markdown 区块

引用

区块

可嵌套
如果在列表中放入区块，需要缩进,例如：

  • 第一项：

    Hello, World!

Markdown 代码

使用反引号来表示代码，例如：
printf("Hello World");

代码区块使用一个Tab，例如:
< ?php
echo Hello World
function test() {
echo test;
}

也可以使用三个反引号来表示一段代码，并指定一种语言，也可以不用指定

a = eval(input())
print(a)

Markdown 链接

链接名称
或者<链接地址>
例如：
Baidu

高级链接

使用变量来设置一个链接，然后在文档末尾对变量进行赋值
例如：百度的网址为Baidu
[1]: www.baidu.com

Markdown 表格

使用 | 来分割不同的单元格，使用 - 来分割表头和其他行
| 表头 |表头 |
| :—-: |:——–:|
|单元格 | 单元格 |

设置对齐

  • -: 设置内容和标题栏右对齐
  • :- 设置内容和标题栏左对齐
  • :-: 设置内容和标题栏剧中对齐
